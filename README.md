# Code Reforge

Code Reforge is a cozy fantasy coding recovery game. This increment is a frontend-only playable vertical slice built with React, TypeScript, and Vite.

## Setup

Install dependencies:

```bash
npm install
```

## Run Locally

Start the Vite development server:

```bash
npm run dev
```

## Build

Create a production build:

```bash
npm run build
```

## Tests

Run the helper tests:

```bash
npm test
```

## Current Scope

- Title screen with new-game and conditional continue actions
- Two-question onboarding flow saved to `localStorage`
- Tavern and quest board screen
- Starter quest mapping based on onboarding answers
- Frontend only: no backend, authentication, AI integration, or code execution
